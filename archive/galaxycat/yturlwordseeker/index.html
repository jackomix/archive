About

Vsauce had a video called "What is Random?" at one part he talked about how sometimes words can random generate in YouTube video URL's. He put text in the description which simply replacing the [your word here] and googling that would see if it existed. I decided for my first web project to create a more simple version of that. Simply type in the word you are looking for in the textbox and press enter. Then see if a YouTube video has that word in it's URL!
